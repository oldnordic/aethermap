<objective>
Implement WASD mode processing for analog stick input, converting 8-way directional output to WASD keyboard keys instead of arrow keys.

Purpose: Enable gamers to use analog stick for movement with WASD keys, which is the standard control scheme for many PC games. The existing D-pad mode uses arrow keys, but many games require WASD.

Output: `wasd_direction_to_keys()` helper function and `process_as_wasd()` method in AnalogProcessor that return Vec<(Key, bool)> for key press/release handling.
</objective>
