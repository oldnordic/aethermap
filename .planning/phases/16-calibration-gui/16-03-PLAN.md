   ```rust
   fn apply_curve(input: f32, curve: &SensitivityCurve, multiplier: f32) -> f32 {
       let curved = match curve {
           SensitivityCurve::Linear => input,
           SensitivityCurve::Quadratic => input * input,
           SensitivityCurve::Exponential { exponent } => {
               if input >= 0.0 {
                   input.powf(*exponent)
               } else {
                   -(-input).powf(*exponent)
               }
           }
       };
       (curved * multiplier).clamp(-1.0, 1.0)
   }
   ```

5. Important: Plot normalized curve (0-1 range) only. Don't show multiplier scaling on graph - multiplier affects output but the graph shows the curve shape. This keeps visualization simple and predictable.
