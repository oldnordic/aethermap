Add broadcast function to aethermap/aethermapd/src/ipc.rs:

1. After handle_request() function, add:
   ```rust
   /// Broadcast analog input update to all subscribers for a device
   pub async fn broadcast_analog_input(
       state: &Arc<RwLock<DaemonState>>,
       device_id: &str,
       axis_x: f32,
       axis_y: f32,
   ) {
       let update = Response::AnalogInputUpdate {
           device_id: device_id.to_string(),
           axis_x,
           axis_y,
       };

       let subscribers = state.read().await.analog_subscribers.clone();
       if let Some(clients) = subscribers.get(device_id) {
           for sender in clients {
               let _ = sender.send(update.clone());
           }
       }
   }
   ```

2. Find the analog event processing location in device.rs or analog_processor.rs.
   Look for where analog events are processed (after process_as_dpad, process_as_gamepad, etc.).

   ```rust
   if let Err(e) = ipc::broadcast_analog_input(&state, &device_id, x, y).await {
       warn!("Failed to broadcast analog input: {}", e);
   }
   ```

Note: This broadcasts the RAW input values (-1.0 to 1.0) before calibration, so the GUI shows actual stick position. The deadzone overlay in visualizer shows how calibration will affect it.

   ```rust
   /// Subscribe to real-time analog input updates for a device
   pub async fn subscribe_analog_input(
       &self,
       device_id: &str,
   ) -> Result<(), String> {
       let request = Request::SubscribeAnalogInput {
           device_id: device_id.to_string(),
       };

       self.send_request(&request).await
   }

   /// Unsubscribe from analog input updates
   pub async fn unsubscribe_analog_input(
       &self,
       device_id: &str,
   ) -> Result<(), String> {
       let request = Request::UnsubscribeAnalogInput {
           device_id: device_id.to_string(),
       };

       self.send_request(&request).await
   }
   ```

Note: These functions follow the same pattern as other IPC functions (get_analog_calibration, set_analog_calibration).
