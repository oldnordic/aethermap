<task type="auto">
  <name>Create layer_manager.rs module with LayerManager, DeviceLayerState, LayerConfig</name>
  <files>razermapper/razermapperd/src/layer_manager.rs</files>
  <action>
Create new file razermapper/razermapperd/src/layer_manager.rs with:

1. LayerMode enum (Hold, Toggle) - tracks how layers activate
   - Hold: Active while modifier key is held
   - Toggle: Press to enable, press again to disable

2. LayerConfig struct:
   - layer_id: usize (0 = base, 1+ = additional layers)
   - name: String (e.g., "Base", "Gaming", "Work")
   - remaps: HashMap<Key, Key> (layer-specific remappings)
   - mode: LayerMode (hold or toggle activation)
   - Implement Default for base layer (layer_id=0, mode=Hold, empty remaps)

3. DeviceLayerState struct:
   - device_id: String (vendor:product format)
   - base_layer: usize (default 0)
   - active_layers: HashSet<usize> (currently active layer IDs)
   - layer_configs: Vec<LayerConfig> (all layer configs, min 3)
   - active_toggle_layers: HashSet<usize> (toggle layers currently active)
   - Methods: new(), get_effective_layer() -> usize, activate_layer(), deactivate_layer(), add_layer_config()

4. LayerManager struct:
   - devices: HashMap<String, DeviceLayerState>
   - Methods: new(), get_device_state(), get_or_create_device_state(), activate_layer(), deactivate_layer(), get_effective_layer()

5. Add #[derive(Debug, Clone, Serialize, Deserialize)] to LayerConfig for IPC
   - Use serde for serialization

Key design decisions:
- Minimum 3 layers supported (indices 0, 1, 2)
- Layer 0 is always the base layer (Hold mode)
- get_effective_layer() returns highest active layer ID, or base_layer if none active
- activate_layer() adds to active_layers HashSet
- deactivate_layer() removes from active_layers HashSet

Use Arc<RwLock<>> pattern from RemapEngine for concurrent access.
