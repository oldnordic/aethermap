  <action>
Update DeviceLayerState::get_effective_layer(&self) -> usize to:

1. Create union of active_hold_layers and active_toggle_layers
2. Find maximum layer ID from the union
3. Return max layer ID if any active, otherwise return base_layer

Implementation approach:
```rust
pub fn get_effective_layer(&self) -> usize {
    let all_active: HashSet<_> = self.active_hold_layers
        .iter()
        .chain(self.active_toggle_layers.iter())
        .copied()
        .collect();

    all_active
        .into_iter()
        .max()
        .unwrap_or(self.base_layer)
}
```

Alternative: Use .max() on iterator without creating intermediate HashSet
(better performance, same result).

This ensures:
- Layer 2 + Layer 1 active = Layer 2 wins (highest ID)
- Hold + toggle layers combine correctly
- No active layers = base_layer returned
