# v1.3 Wayland & Analog Advanced - Milestone Summary

**Created:** 2026-02-18
**Status:** READY FOR APPROVAL

## Milestone Overview

**v1.3 Wayland & Analog Advanced** adds two major feature areas to Aethermap:

1. **Wayland Integration** - Auto-profile switching based on window focus via xdg-desktop-portal, plus global hotkeys that work on Wayland compositors
2. **Advanced Analog Modes** - Gamepad emulation (Xbox 360), WASD output, mouse control, and camera/scroll modes with full calibration GUI

## Success Criteria Preview

### Phase 13: Wayland Portal Integration
1. GUI detects window focus changes via xdg-desktop-portal
2. Focus changes trigger profile switches based on app_id rules
3. Global hotkeys (Ctrl+Alt+Shift+Number) work on Wayland
4. Portal permission errors handled gracefully with fallback
5. Auto-switch rules configurable in YAML

### Phase 14: Gamepad Emulation Mode
1. Analog stick outputs to virtual Xbox 360 gamepad device
2. Games recognize device via standard Linux drivers
3. Calibration affects gamepad output
4. Per-layer gamepad mode configuration supported

### Phase 15: WASD, Mouse, Camera Modes
1. WASD mode outputs 8-way directional keys with diagonals
2. Mouse mode moves cursor with velocity-based control
3. Camera mode emits scroll or key repeat events
4. Mode switchable via GUI and IPC, per-layer configurable
5. Smooth transitions between analog directions

### Phase 16: Calibration GUI
1. Real-time analog input visualization (stick position)
2. Deadzone controls (slider, shape selector, visual preview)
3. Sensitivity controls (curve selector, multiplier, graph)
4. Range controls (min/max sliders, inversion)
5. All changes apply immediately with live preview
6. Calibration persists to device profile YAML

## Technical Stack

**Wayland Integration:**
- ashpd 0.8 (Rust xdg-desktop-portal bindings)
- zbus 3.0 (D-Bus for portal communication)
- No changes to daemon privileges (portal access from GUI only)

**Analog Modes:**
- Extend existing AnalogProcessor from Phase 11
- evdev::uinput for virtual gamepad device
- Strategy enum pattern for mode switching

**Calibration GUI:**
- Iced 0.12 (existing GUI framework)
- Canvas widgets for real-time visualization
- IPC subscription for live analog input
